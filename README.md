# master_thesis

## Evaluating Sarcasm Preservation in Machine Translation from English to Russian and Its Impact on Automatic Sarcasm Detection

This repository contains the code and experiments accompanying my Master's thesis. The study investigates how Machine Translation (MT) affects the preservation of sarcastic meaning and how these changes influence the performance of automatic sarcasm detection models in the English-Russian language pair.

The notebook includes:
- data preprocessing;
- machine translation;
- 4 fine-tuning configurations;
- evaluation of model performance;
